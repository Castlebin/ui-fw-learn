# Vue 2.x 学习

## 安装 Vue

1. 建议在浏览器中安装 Vue Devtools，便于开发调试
2. 使用 Vue 的几种方法：
   1）普通 HTML 页面，直接用 \<script\> 标签 引入
   2）使用 NPM 构建应用， 命令 npm install vue 安装 vue 到本地
   3）使用 vue-cli 快速搭建项目框架
